# Demo Discussions

These demo discussions define the first content examples for local seed data, UI states, tests, and screenshots. They are deliberately chosen to exercise different parts of the discourse model.

## Demo 1: Wärmepumpen

Topic path:

```text
Energie
  -> Gebaeude
    -> Wärmepumpen
```

Core question:

```text
Sind Wärmepumpen im Gebäudebestand sinnvoll?
```

Positions:

- Ja, Wärmepumpen sind im Gebäudebestand sinnvoll.
- Nur bei passenden baulichen Voraussetzungen.
- Nein, sie sind im Bestand oft unwirtschaftlich.

Seed Claims:

- Wärmepumpen sind effizient.
- Die Effizienz haengt stark von Vorlauftemperatur und Gebaeudehuelle ab.
- Strommix und Betriebskosten beeinflussen die Klimabilanz.
- Unsanierte Altbauten koennen die Wirtschaftlichkeit verschlechtern.

Argument and Evidence goals:

- Show support and attack chains.
- Show a Claim that is repeatedly challenged and defended.
- Show source verification for official data, studies, and real-world reports.

## Demo 2: Feminismus

Topic path:

```text
Gesellschaft
  -> Feminismus
    -> Gleichberechtigung im Beruf
```

Core question:

```text
Welche Massnahmen verbessern Gleichberechtigung im Beruf am wirksamsten?
```

Positions:

- Transparente Gehaltsstrukturen sind am wirksamsten.
- Ausbau von Kinderbetreuung und Elternzeit ist zentral.
- Anti-Bias-Prozesse bei Einstellung und Beförderung sind wichtiger.

Seed Claims:

- Gehaltstransparenz reduziert strukturelle Ungleichheit.
- Elternzeitmodelle beeinflussen Karriereverläufe.
- Bias-Trainings allein reichen nicht aus.
- Gesetzliche Vorgaben wirken anders als freiwillige Unternehmensprogramme.

Argument and Evidence goals:

- Exercise nested Topics.
- Exercise normative and factual Claims.
- Show how one Argument can be relevant in multiple Discussions.

## Demo 3: Pianouniverse

Topic path:

```text
Pianouniverse
  -> Lernen
    -> Ueben
```

Core question:

```text
Soll Pianouniverse strukturierte Diskurse zu Uebemethoden anbieten?
```

Positions:

- Ja, Diskurse helfen Lernenden bei reflektierten Entscheidungen.
- Nur in Kurs- oder Lehrer-Spaces.
- Nein, die Plattform sollte sich auf Unterrichtsinhalte konzentrieren.

Seed Claims:

- Lernende profitieren von begruendeten Erfahrungsberichten.
- Oeffentliche Diskussionen koennen didaktische Inhalte verwässern.
- Private Kurs-Spaces reduzieren Moderationsrisiken.
- Evidence kann hier aus Studien, Lehrpraxis und Nutzererfahrungen bestehen.

Argument and Evidence goals:

- Exercise future integration with Pianouniverse.
- Exercise private Spaces and public default assumptions.
- Show how experiential Evidence differs from scientific Evidence.
